## Install

``` shell
git clone https://github.com/xyzhanjiang/react-bulma-admin.git
cd react-bulma-admin
npm install
```

## Usage

**Develop**

``` shell
npm run start
```

Visit localhost:3000

**Build**

``` shell
npm run build
```

## Structure

``` diff
bulma-admin
|- package.json
|- /public
  |- index.html  // home page
|- /src
  |- app.js      // the main application component
  |- index.js    // the entry point file for the application
  |- /common
  |- /components
  |- /config
  |- /css        // styles
  |- /data
  |- /images
  |- /store      // redux store
  |- /views      // page views
    |- about.js  // about page
    |- index.js  // index page
    |- login.js  // login page
```
